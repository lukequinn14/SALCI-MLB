# ⚾ SALCI - Strikeout Adjusted Lineup Confidence Index

A MLB prediction model for pitcher strikeouts and hitter matchups.

## What is SALCI?

SALCI combines pitcher metrics with opponent tendencies to predict strikeout performance:

### Pitcher Metrics (60%)
- **K/9** - Strikeouts per 9 innings
- **K%** - Strikeout rate
- **K/BB** - Strikeout to walk ratio
- **P/IP** - Pitches per inning (efficiency)

### Matchup Factors (40%)
- **Opp K%** - Opponent team's strikeout rate
- **Opp Contact%** - Opponent's contact rate

## Features

- 🎯 **Pitcher Analysis** - SALCI scores and K-line projections
- 🏏 **Hitter Matchups** - Hot/cold streaks, platoon advantages
- 📊 **Shareable Charts** - Built for Twitter/X posts
- 📱 **Game Day Card** - Daily summary graphic
- ✅ **Confirmed Lineups** - Only shows actual starters

## SALCI Rating Scale

| Score | Rating | Meaning |
|-------|--------|---------|
| 75+ | 🔥 Elite | Top-tier K potential |
| 60-74 | ✅ Strong | Above average |
| 45-59 | ➖ Average | Coin flip |
| 30-44 | ⚠️ Below Avg | Fade territory |
| <30 | ❌ Poor | Stay away |
